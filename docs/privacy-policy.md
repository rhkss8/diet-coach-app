# Privacy Policy Draft

This is a draft for closed MVP testing and must be reviewed before public release.

## Data We Collect

- basic profile inputs such as age, sex, height, and current weight
- goal inputs such as target weight and target date
- diet and exercise plan items
- manual adjustment reasons and notes
- plan revision history
- check-in and completion status
- product analytics events

## How We Use Data

We use this data to create, adjust, and improve the user's diet plan. The MVP focuses on helping users continue after real-life plan changes.

## AI Processing

AI outputs are used to generate and adjust plans. AI output is reviewed by the user before it changes persisted plan data.

## What This App Is Not

The app is not medical advice, diagnosis, or a precise calorie tracking tool.
