# Superpowers

## Definition
Superpowers are the few abilities Diet Planner gives the user that existing products do not.

## MVP Superpowers
### 1. Adjust Today In Seconds
As a user, I can adjust today's plan in seconds when real life changes, so that I do not have to restart my diet.

Product flow:
- Today screen.
- "Adjust today" button.
- Reason selection.
- Revised plan proposal.

Test:
- A fresh user can find and complete adjustment without free-form chat.

### 2. Stay In Control
As a user, I can approve changes before they affect my plan, so that I stay in control.

Product flow:
- Revision review screen.
- Approve or dismiss revision.
- Persist only approved revisions.

Test:
- Dismissed revisions do not change the plan.

### 3. Continue Without Shame
As a user, I can see that adjusted days still count as continuation, so that I do not feel like I failed.

Product flow:
- Today status.
- Progress screen.
- Revision history.

Test:
- Copy and metrics avoid failure language.
