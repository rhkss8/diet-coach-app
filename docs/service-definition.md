# Service Definition

## One-Line Definition

Diet Planner helps people whose diet plans break in real life continue without restarting by making manual plan adjustment fast, calm, and reviewable.

## Persona

A person who wants to lose weight but cannot follow rigid diet plans because meals, work, energy, and schedule change often.

## Core Pain

Existing diet apps turn a changed day into failure. The user wants to continue, not confess failure.

## Desired Transformation

Before:

- The user feels that one broken day ruins the whole plan.

After:

- The user feels they can adjust today and keep going.

## Core Mechanism

Manual plan adjustment:

- User asks to adjust.
- App proposes a revised plan.
- User approves.
- The plan continues from the revised state.
