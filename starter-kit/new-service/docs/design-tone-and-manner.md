## Visual Direction

Recommended:

- Clear mobile-first layouts.
- Strong hierarchy.
- Familiar controls.
- Visual style that matches the user's real context.

Avoid:

- Decorative UI that does not support the task.
- Marketing-style hero pages inside the product.
- Generic AI chatbot styling unless chat is truly the core interaction.
- UI patterns that conflict with the product's tone.
